#TCS Financial Analysis (2019–2026)

##Overview

In this project, I analyzed the financial performance of Tata Consultancy Services (TCS) using Microsoft Excel.
The analysis is based on key financial metrics such as revenue, net profit, and operating profit over the period from 2019 to 2026.

##Objective

The main objective of this project was to understand how a large IT company like TCS has performed financially over the years and to identify trends in its growth and profitability.

##Tools Used

Microsoft Excel (for data analysis and visualization)

##Analysis Performed

Revenue trend analysis
Net profit trend analysis
Operating profit analysis
Operating margin (OPM) observation

## Key Insights

-TCS has shown consistent revenue growth from 2019 to 2026
-Net profit has increased steadily, indicating strong profitability
-Operating profit has also grown, showing efficient operations
-Operating margin has remained stable around 26–28%
-Growth has slightly slowed in recent years, suggesting a mature stage

##Conclusion

Based on the analysis, TCS appears to be a financially strong and stable company with consistent performance over the years.
